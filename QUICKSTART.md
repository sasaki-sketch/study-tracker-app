# クイックスタートガイド

## 5分で始める診断士学習記録アプリ

### ステップ1: インストール（2分）

```bash
# 1. ディレクトリに移動
cd ~/study_app

# 2. 依存パッケージをインストール
pip3 install -r requirements.txt

# 3. データベースを初期化
python3 database/init_db.py
```

### ステップ2: 動作確認（1分）

```bash
# テストを実行
python3 test_app.py
```

以下のメッセージが表示されればOK:
```
✅ すべてのテストが正常に完了しました！
```

### ステップ3: アプリ起動（1分）

```bash
# Streamlitアプリを起動
streamlit run app.py
```

ブラウザで `http://localhost:8501` が自動的に開きます。

### ステップ4: 最初の記録を入力（1分）

1. **日付**: 今日の日付を選択（デフォルトで今日）
2. **診断士セクション**:
   - 学習時間: `3` (h)
   - 科目: `財務会計`
   - 学習内容: `過去問15問 正答率70%`
   - 課題: `固変分解の理解`
3. **統計セクション**:
   - 学習時間: `1` (h)
   - 学習内容: `推定演習`
   - 課題: `信頼区間の理解`
4. **「💾 保存してX投稿文生成」をクリック**

## Raycastから起動する（オプション）

### 設定方法

1. Raycastを起動: `⌥Space`
2. コマンド: `Import Script`
3. ファイル選択: `~/raycast-scripts/study-app-open.sh`
4. インポート完了

### 使い方

1. `⌥Space` でRaycast起動
2. `学習記録` と入力
3. Enter → アプリが自動起動

## よくある質問

### Q: アプリを終了するには？

**A:** ターミナルで `Ctrl+C` を押す、またはブラウザタブを閉じる

### Q: ポート8501が使用中と表示される

**A:** 別のポートで起動:
```bash
streamlit run app.py --server.port 8502
```

### Q: データをリセットしたい

**A:** データベースを削除して再初期化:
```bash
rm ~/study_app/study_records.db
python3 database/init_db.py
```

### Q: Obsidianファイルが作成されない

**A:** 出力先ディレクトリを作成:
```bash
mkdir -p ~/Documents/01_Knowledge/obsidian-vault/03_Projects/診断士2026_一発合格/09_学習記録/
```
